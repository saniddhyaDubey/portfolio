# saniddhya dubey — personal portfolio

This is my personal portfolio website, built with Next.js. It is not a template and is not intended for forking or public contributions.

## stack

- **Next.js** (App Router)
- **Tailwind CSS**
- **Framer Motion**
- **next-themes**

## pages

- `/` — home
- `/logs` — daily notes, small wins, tech learnings
- `/oss` — open source contributions
- `/about` — about me
- `/portfolio` — projects
- `/contact` — get in touch
